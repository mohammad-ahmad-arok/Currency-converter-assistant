# مساعد تحويل العملة - Currency Transition Helper

تطبيق ويب بسيط وواضح لمساعدة الناس في سوريا خلال فترة تغيير العملة.

## المميزات

- ✅ حاسبة تحويل العملة
- ✅ عداد النقود (عد الأوراق النقدية)
- ✅ عرض أسعار الصرف
- ✅ مقارنة قبل وبعد التغيير
- ✅ سيناريوهات جاهزة (راتب، إيجار، دين، سعر منتج)
- ✅ دعم PWA (يمكن تثبيته كتطبيق)
- ✅ يعمل بدون إنترنت
- ✅ واجهة بسيطة وسهلة الاستخدام
- ✅ تصميم متجاوب للهواتف المحمولة
- ✅ وضع النص الكبير لسهولة القراءة

## التقنيات المستخدمة

- HTML5
- CSS3 (تصميم متجاوب)
- JavaScript (Vanilla JS)
- Progressive Web App (PWA)
- Service Worker (للعمل بدون إنترنت)
- LocalStorage (لحفظ الحسابات)

## كيفية الاستخدام

1. افتح `index.html` في المتصفح
2. أو استضف الملفات على خادم ويب
3. للتثبيت كتطبيق: اضغط على زر الإعدادات ⚙️ ثم "تثبيت التطبيق"

## الملفات

- `index.html` - الصفحة الرئيسية
- `styles.css` - التنسيقات والتصميم
- `app.js` - منطق التطبيق
- `manifest.json` - ملف PWA
- `sw.js` - Service Worker للعمل بدون إنترنت

## ملاحظات

- الأسعار في التطبيق إرشادية فقط
- يمكن تعديل أسعار الصرف في ملف `app.js` في متغير `appState.exchangeRates`
- التطبيق يحفظ آخر 50 عملية حسابية محلياً

## الأيقونات

يحتاج التطبيق إلى أيقونات بحجم 192x192 و 512x512 بكسل. يمكنك إضافة:
- `icon-192.png`
- `icon-512.png`

أو تعديل `manifest.json` لإزالة الإشارة إليها.

## الترخيص

صُنع بحب لمساعدة الناس في سوريا.
